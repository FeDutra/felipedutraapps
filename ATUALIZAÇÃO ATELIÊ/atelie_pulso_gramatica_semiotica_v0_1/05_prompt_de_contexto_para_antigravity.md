# Contexto para Antigravity — Ateliê PULSO / Gramática Semiótica v0.1

Estou enviando uma pasta com documentos para orientar a próxima etapa do Ateliê da PULSO.

A ideia central é que o Ateliê não seja um editor de efeitos, mas um campo de relações semióticas.

Regra principal:

```text
elemento + alvo = modulação
elemento + elemento = composto
composto + alvo = atuação híbrida
```

Ou seja:

- se um elemento encosta em um som, o som reage;
- se um elemento encosta em uma imagem, a imagem sofre interferência;
- se dois elementos estão próximos entre si, nasce uma terceira assinatura composta;
- se esse composto toca som ou imagem, a atuação deve ser híbrida, não dois presets isolados.

Arquivos principais:

1. `01_gramatica_de_atuacao_semiotica_v0_1.md`
   - Princípios, regras de proximidade, dominância, composição e incidência.

2. `02_tabela_atuacao_som_v0_1.md`
   - Primeira tabela conceitual para atuação em áudio.

3. `03_tabela_atuacao_imagem_v0_1.md`
   - Primeira tabela conceitual para atuação visual/distorção de imagem.

4. `04_modelo_dados_para_implementacao_v0_1.json`
   - Um modelo inicial de estrutura de dados. Não é final, mas serve como seed.

5. `99_tabela_original_antigravity.md`
   - Tabela original, para comparação e preservação do que já existe.

Tarefa esperada:
ler esse material e propor como transformar isso em estrutura de dados e implementação gradual no Ateliê.

Não precisa implementar tudo de uma vez.

Prioridades:

1. preservar o que já funciona no som;
2. diferenciar melhor as atuações por elemento, especialmente emoções e subemoções;
3. criar atuação visual real para imagem, não apenas overlay;
4. permitir combinação de elementos por proximidade;
5. evitar presets isolados demais;
6. preparar estrutura para que som e imagem compartilhem a mesma gramática semiótica.

Importante:
as referências visuais enviadas são só vocabulário de possibilidade: ASCII, dithering, halftone, pixel sort, contour, scanline, point cloud, aberração cromática, glitch, rasterização, deslocamentos, máscaras e ruídos. Não é para copiar uma estética específica.
