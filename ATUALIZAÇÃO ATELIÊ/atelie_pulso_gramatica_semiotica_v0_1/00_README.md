# Ateliê PULSO — pacote de gramática semiótica v0.1

Pacote para enviar ao Antigravity.

Objetivo: transformar a tabela atual do Ateliê em uma gramática de atuação semiótica para som e imagem.

Arquivos:

- `00_README.md`: visão geral do pacote.
- `01_gramatica_de_atuacao_semiotica_v0_1.md`: princípios e regras de relação.
- `02_tabela_atuacao_som_v0_1.md`: tabela revisada para atuação sonora.
- `03_tabela_atuacao_imagem_v0_1.md`: tabela revisada para atuação visual.
- `04_modelo_dados_para_implementacao_v0_1.json`: modelo técnico inicial.
- `05_prompt_de_contexto_para_antigravity.md`: texto curto para acompanhar o envio.
- `99_tabela_original_antigravity.md`: tabela original recebida do Antigravity, para comparação.

Regra central:

```text
elemento + alvo = modulação
elemento + elemento = composto
composto + alvo = atuação híbrida
```

Ou seja, o elemento não aplica apenas um preset. Ele cria relação, contaminação, tensão, fusão, instabilidade ou amplificação.
